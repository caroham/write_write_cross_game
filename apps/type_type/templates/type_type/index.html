Write Write Cross

A game where friends or strangers can write something interesting or silly together.

How to play

  • Invite a friend or rando to start a project
  • Players take turns adding or deleting words
  • For every two words that are added, one word must be deleted
  • Example: writer1 writes a word, writer2 adds a word, writer1 must then delete one of the two existing words. This cycle continues as long as you want it to!
